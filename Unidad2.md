### Estructuras condicionales (tipos y ejercicios en diagrama de flujo y en C)
Esta tipo de estructura evalúa una condición lógica (Verdadero o Falso)
para decidir qué bloque de código ejecutar. Permite que el programa tome decisiones 
y siga diferentes caminos. Los tipos de Estructuras condicionales son: [1].

**Estructura Condicional Simple:**
Esta estructura recibe un parámetro el cual se ejecutará dependiendo si el valor es
verdadero, si este se cumple ejecutara el bloque de código que esta en ejecución, y si no
cumple se salta el bloque de código [2].

**Estructura condicional doble:**
Esta estructura recibe un parámetro el cual le devuelve a un valor booleano, ya que es una
expresión de comparación, si la condición se cumple ira a la derecha ejecutando la parte
verdadera, y si no se cumple ira a la izquierda y se ejecutan la parte falsa [2].

Estructura Condicional Múltiple:
Esta estructura se encarga de evaluar el dato que ingresa después de un procedimiento o,
de entrada, a la vez que esta estructura solo puede leer los siguientes parámetros.
Booleanos, char (o cadena), tipos numéricos y enumeración.
Así mismo algo a recalcar es que está conformado por etiquetas que permiten comprobar
cada caso y si ninguno se comprueba dará un default [2].
